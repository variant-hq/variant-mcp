# Security Policy

## Reporting Security Issues

Do not open public issues for vulnerabilities, leaked credentials, private deck content, or account-specific access problems.

Report security issues through Variant's private support channel from the app or website.

## Credential Handling

- Never commit Variant MCP bearer tokens.
- Do not paste OAuth tokens, bearer tokens, deck contents, or private prompts into GitHub issues.
- Use the smallest credential scope that fits the workflow.
- Revoke tokens that are no longer needed.
