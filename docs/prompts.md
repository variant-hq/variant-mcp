# Prompt Examples

These prompts are written for MCP clients connected to Variant.

## Read-Only Smoke Test

```txt
Connect to Variant, list my decks, choose the most recently updated deck, list its slides, and preview slide 1. Do not edit anything.
```

## Create A Deck

```txt
Create a six-slide Variant deck for a Q3 launch review. Use concise executive copy, include speaker notes, and preview the title slide when done.
```

## Edit One Element

```txt
In my launch deck, change only the headline on slide 4 to "Launch readiness is improving". Use slide.edit and do not rewrite the rest of the slide.
```

## Use Brand Context

```txt
Read my Variant brand context before drafting. Use the logo, colors, typography guidance, and voice notes from the brand kit.
```

## Export

```txt
Export this Variant deck as PPTX for the client handoff and as HTML for internal review.
```

## Review Without Editing

```txt
Review this deck for unclear slide titles, repeated claims, and missing evidence. Do not edit the deck.
```
